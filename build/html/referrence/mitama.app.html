class mitama.app.builder.Builder¶

ベースクラス: object

Appにメタ情報を入力して生成するビルダー

プロジェクトディレクトリのパスやインストール先、プロジェクト内のアプリ用ディレクトリのパスをAppに設定し、インスタンスを返却します。 特にこれを弄るケースは想定していませんが、独自の挙動を付けたかったら継承して作っても良いかもしれません。 アプリのパッケージ直下のAppBuilderが起動されるので、__init__.py に class AppBuilder(Builder) を定義してください。
